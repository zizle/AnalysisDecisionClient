注1:初价 =（原糖价格＋升贴水）x 重量单位换算 x 旋光度增值
注2:海运保险费= 初价 x 海运保险费率
注3:到岸价(人民币) = (初价 + 海运保险费 + 海外运费) x 汇率
注4:不含税价 = 到岸价 x (1 + 外贸代理费率) x (1 + 银行手续费率) + 利息及劳务费
注5:含税价= 不含税价 x (1 + 海关税率) x (1 + 增值税率)
注6:精炼后完税价格 = 含税价 x(1+0.07) + 加工费 + 国内运费。
注7:结果四舍五入到小数点后4位
